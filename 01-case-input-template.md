# 01. 케이스 입력 양식

## 최소 입력

사용자는 모든 환자 정보를 줄 필요가 없습니다. 발표 주제와 관련된 핵심만 주면 됩니다.

```md
## 케이스
-

## 수술/치료 포인트
-

## 교수님이 강조한 내용
-

## 사용자가 궁금한 점
-
```

## 예시

```md
## 케이스
Distal radius fracture, dorsal Barton fracture malunion

## 수술/치료 포인트
Double plating으로 dorsal angulation 회복

## 교수님이 강조한 내용
없음

## 사용자가 궁금한 점
없음. 발표 주제와 강조 질문을 먼저 추천해줘.
```

## 제외해도 되는 정보

발표 주제와 직접 관련이 없다면 아래 정보는 생략할 수 있습니다.

- 환자 나이
- 성별
- 자세한 수상기전
- 입원 경과
- 모든 과거력
- 모든 검사 수치

단, 아래 정보가 발표 주제와 직접 연결되면 포함합니다.

- 치료 결정에 영향을 준 나이/활동도
- 수상기전이 병태생리를 설명하는 경우
- 추적 결과가 문헌고찰 주제와 연결되는 경우
- 합병증 또는 실패 원인이 핵심 주제인 경우

## 입력이 부족할 때 처리

정보가 부족해도 작업을 멈추지 않습니다.

먼저 가능한 발표 주제 후보를 제안하고, 필요한 경우에만 추가 질문을 합니다.

추가 질문은 1-3개만 합니다.
